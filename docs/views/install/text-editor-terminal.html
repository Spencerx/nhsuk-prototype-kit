Text editor and terminal

You need a text editor and terminal to edit and make changes to your prototype.

For a text editor we recommend using Visual Studio Code opens in new window. It's free and includes an integrated terminal.

Terminal

The terminal lets you type in commands and run programs on your computer.

To use the integrated terminal, open up Visual Studio Code and go to View > Integrated Terminal.
